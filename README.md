# Taller de listas

- taller para aprender sobre como usar las distintas listas que existen en python.

## Ejercicio #1

crear una lista de tres de mis mejores amigos, en donde agrego uno y expulso a otro, esto con los comandos ".append" para agregar y ".pop" para sacar y eliminar de la lista.

## Ejercicio #2

crear dos listas, una con  las edades de compañeros de el salon y otra con los gustos musicales, y con estos datos, sacar el promedio de edad y cantidad de fanes del rock.
